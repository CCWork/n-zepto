# n-zepto
Zepto for npm.
Based Zepto 1.2.0(The latest version)

## Install

Install via npm:

`npm install n-zepto --save`

Install via yarn:

`yarn add n-zepto`

## Usage

```javascript
import $ from 'n-zepto'
```
